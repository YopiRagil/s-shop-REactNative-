# s-shop-REactNative-

For installing node module run

```bash
yarn install
```

For run in mode development run:

```bash
expo start
```

or

```bash
yarn web
```

for build to apk run:

```bash
expo build:android
```

for try apk that I build you can download apk file [here](https://expo.io/artifacts/6abafa9c-edce-4704-9468-8438356b6951)
